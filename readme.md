# ASMIN - Restful API
> Asmin A.K.A Aspire mini is a mini aspire-cap.

## Requirements and dependencies

- PHP >= 5.6^
- Lumen 5.4

## How it works!

```
$ git clone https://github.com/harunnryd/asmin.git
$ composer install
```
